# 产品说明书

## 系统名称
Motila 综合关系系统

## 本次功能更新（2026-03-10）

### 1. 新增「配置 -> 系统配置」
新增系统配置管理页面，支持：
- 系统名称（name）
- 系统标题（title）
- 系统 Logo 图片地址（logoUrl）
- 页脚文案（footerText）
- 是否有效（isActive）

### 2. 生效规则
- 同一时间仅允许 **1 条有效配置**（`isActive=true`）。
- 当某条配置被设置为有效时，其他配置会自动失效。

### 3. 前端联动
当存在有效系统配置时：
- 左侧 Logo 区域：优先显示 Logo 图片；无图片时显示系统名称
- 顶部标题：显示系统标题
- 浏览器标签标题（document.title）：同步系统标题
- 页面底部 Footer：显示系统页脚文案

无有效配置时，使用默认值：
- Logo 文案：`Motila`
- 标题：`Motila 管理系统`
- Footer：`Motila © 2026`

### 4. 权限控制
新增权限点：
- `settings.read`：查看系统配置
- `settings.update`：修改系统配置

说明：未授权用户无法访问 `/settings/system`。

### 5. Logo 生效优先级（2026-03-10 补充）
系统展示 Logo 的优先级调整为：
1. 上传图片（`logoImage`，base64）
2. `logoUrl`
3. 默认文案 `Motila`


### 7. 顶部用户信息展示优化（2026-03-11）
- 右上角用户信息由按钮改为“头像 + 昵称 + 角色”的信息块展示。
- 点击信息块仍可展开个人中心/修改密码/退出登录菜单。
