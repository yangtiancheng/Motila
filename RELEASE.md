# RELEASE.md

> 项目：Motila
> 维护方式：每次迭代结束更新「已完成」与「下一迭代计划」
> 更新时间：2026-03-15

- 未登录首页重构为极简终端风格：默认展示 `Welcome to Motila.` / `Hello World.` 动效
- 登录/注册页从首页分离为独立路由：`/login`、`/register`
- 认证页切换登录/注册时同步更新浏览器地址，支持直接访问与刷新保留页面
- 首页顶部新增「博文 / 文档 / 社区」预留入口，当前点击统一提示“暂未开放，先别急。”
- 博客管理拆分为「博客分类 / 博客文章」两套菜单与列表页面
- 博客分类/文章创建与编辑改为独立页面（与人员管理一致），不再使用弹窗
- 修复博客模块依赖与路由拆分后菜单访问规则
- 新增风控状态重置能力：支持按 IP 或按用户/账号清理运行期风控计数与封禁（可选场景）
- 优化用户头像编辑体验：头像上传预览改为圆形显示风格
- 优化系统内置角色编辑限制：`DEFAULT_USER` / `ADMIN` 等系统角色仍不可修改元信息、权限、模块，但允许调整“包含用户/分配用户”多选字段
- 前端登录页 / 注册页 / 忘记密码弹窗新增验证码展示区、输入框与“换一张”交互
- 优化 forgot-password 安全策略：不再暴露账号是否存在，统一返回泛化提示文案
- 优化风控降级行为：Redis 不可用且策略为 `ALLOW_WITH_CAPTCHA` 时，运行期回退到内存计数，便于开发环境验证风控链路
- 新增风控配置中心：支持在“配置 -> 风控配置”中统一维护登录 / 注册 / 忘记密码场景策略
- 新增风控配置版本能力：支持草稿保存、发布生效、历史版本回滚
- auth 接口接入风控运行层：登录 / 注册 / 忘记密码支持按 IP / 账号进行限频与失败封禁
- 新增 Redis 运行依赖：支持 `REDIS_URL` / `REDIS_HOST` 配置，Redis 不可用时按降级策略执行
- 新增权限点：`risk.read`、`risk.update`
- 移动端导航优化：手机端新增顶部菜单按钮 + Drawer 抽屉菜单，解决左侧菜单自动折叠后无法选择的问题
- 菜单交互优化：移动端点击菜单项后自动关闭抽屉，避免遮挡内容
- API 请求基址统一为相对路径 `/api`，为 HTTPS + 反向代理场景打通前后端联调
- 配置页优化：系统配置“编辑/新建”由弹窗改为页面内标准编辑区（Card）
- 修复登录页标题首屏闪烁：`index.html` 默认标题从 `frontend` 调整为 `Welcome to Motila`
- 新增系统配置 Logo 上传能力（png/jpg/webp），并支持保存为 `logoImage`
- Logo 展示优先级调整：`logoImage` > `logoUrl` > 默认文案
- 修复系统配置创建时 `logoUrl` 为空触发 URL 校验报错问题
- 修复系统配置创建时报错 `property removeLogoImage should not exist`：前端仅在编辑(PATCH)时提交 `removeLogoImage`，新建(POST)不再携带该字段
- 新增个人信息页邮箱配置（QQ/163）：支持 SMTP/IMAP 主机端口、授权码保存（加密）与测试按钮
- 新增登录页“忘记密码”邮箱找回：支持用户名或邮箱触发，发送临时密码邮件并要求登录后修改
- 个人信息页支持直接修改昵称/用户名/头像地址，并去除重复邮箱展示（邮箱维护统一在邮箱配置区）
- 用户管理编辑页修复：用户名可正确回显；用户名修改保持唯一性校验
- 用户管理编辑布局优化：主字段按4列展示，头像相关字段按2列展示
- 未登录首页升级为公开博客首页：接入已发布博客文章展示，支持分类筛选、搜索、分页和美观详情页，Footer 按系统配置文案显示
- 修复基础页面框架样式：创建/编辑等带操作区（Card extra）的页面头部默认可见，避免保存/取消按钮被全局样式隐藏
- 用户管理列表移动端适配优化：表格支持横向滚动，避免小屏内容溢出
- 用户头像支持上传图片（base64）与预览，展示优先级：avatarImage > avatarUrl
- 修复系统配置 logoImage 未持久化导致左上角 Logo 不显示的问题
- 修复权限配置中分配用户报错：`pageSize must not be less than 1`（支持 `pageSize=0` 用于全量拉取）
- 顶部用户区域改为“头像 + 昵称 + 角色”展示（非按钮样式）
- 用户管理新增头像字段（avatarUrl），列表与详情页展示
