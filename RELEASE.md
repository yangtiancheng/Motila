## v0.1.0（当前）

### 已完成功能清单

- 前后端工程初始化（Vite React + NestJS + Prisma）
- 登录后白屏修复（前端入口补充 BrowserRouter，修复路由上下文缺失）
- 认证模块：注册 / 登录 / JWT
- 首个用户自动授予 ADMIN
- 用户管理 API：查询/创建/编辑/删除（管理员）
- 用户管理 UI：列表、新建、编辑、删除、角色切换
- 登录/注册页视觉升级（卡片化、渐变背景、响应式）
- 皮肤系统（可持续迭代）
  - 商务蓝（business）
  - 科技紫（tech）
  - 暗夜黑（dark）
  - 跟随系统（auto，根据系统深浅色自动切换）
  - 支持通过 `theme-skins.ts` 统一扩展主题和配色
  - 品牌色配置入口（ColorPicker，支持实时换主色）
  - 主题配置导入/导出（JSON，可跨环境复用）
  - 皮肤选择持久化（localStorage）
- 安全策略：
  - 禁止删除当前登录用户
  - 禁止将自己降权为 USER
  - 禁止删除/降权最后一个 ADMIN
- 开发数据库切换为 SQLite（`backend/dev.db`）
- 一键启动脚本：`npm run dev:all`
- PM2 守护进程托管前后端
- PM2 开机自启（systemd）

### 运维可用性

- 前端可访问：`http://172.21.40.8:5173`
- 后端健康检查：`http://172.21.40.8:3000/health`
